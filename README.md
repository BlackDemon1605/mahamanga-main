# Maha Manga

A manga/comic reading platform built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Browse and read manga, manhwa, and webtoons
- User authentication and profiles
- Bookmarking and reading history
- Rating and commenting system
- Creator uploads and management

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (Backend)
